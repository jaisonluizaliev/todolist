*O QUE É O MONGODB*

O MONGODB É UM BD ORIENTADO A DOCUMENTOS

ELE ARMAZENA DOCU EM FORMATO similar a JSON  QUE NÃO PRECISA TER UMA ESTRUTURA PRE DEFINIDA

EXEMPLO SERIA ISSO

{
  "bairro":"tapera",
  "area":"20kmº"
  "numero de habitantes:"30000"
  "cadastro de moradores": [
    {
      "nome":"jaison"
      "age": 26
    },
    {
      "nome":"sabrina"
      "age": 23
    }
  ],
}

*BSON*

O MONGO INTERNAMENTE REPRESENTA OS DOCUMENTOS JSON EM UM FORMATO CHAMADO BSON QUE É UMA VERSÃO BINÁRIA-ENCODADA DO JSON PARA INCLUIR ALGUNS TIPOS EXTRA DE FORMATOS  DE DADOS E PARA CODIFICAÇÃO EM LINGUAGENS DIFERENTES
